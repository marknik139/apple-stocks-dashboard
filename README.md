## Getting Started

To get a local copy up and running follow these steps

### Prerequisites

Install Node.js
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/marknik139/apple-stocks-dashboard.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run
   ```sh
   npm start
   ```
